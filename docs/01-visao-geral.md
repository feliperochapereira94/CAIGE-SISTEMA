# Visao Geral

## Apresentacao do projeto

O CAIGE e um sistema web voltado ao gerenciamento de informacoes do Centro de Atencao Integral ao Idoso. A aplicacao foi desenvolvida para apoiar o registro e a consulta de dados de pacientes, frequencia, prontuarios, questionarios e administracao de usuarios.

## Problema resolvido

O projeto busca centralizar informacoes operacionais e clinicas em um unico sistema, reduzindo o uso disperso de registros manuais e facilitando o acompanhamento das atividades realizadas no contexto academico e assistencial do CAIGE.

## Objetivo geral

Disponibilizar uma plataforma web para cadastro, consulta e administracao de dados relacionados aos atendimentos realizados no CAIGE.

## Objetivos especificos

- cadastrar e manter os dados dos pacientes;
- registrar frequencia e presenca em atendimentos;
- armazenar prontuarios e anexos por especialidade;
- aplicar questionarios vinculados a cursos e pacientes;
- controlar acesso de usuarios por papel;
- apoiar a gestao por meio de dashboard e registros de atividades.

## Publico-alvo

- supervisores responsaveis pela administracao do sistema;
- professores que realizam atendimento, registro e acompanhamento;
- equipe tecnica responsavel pela manutencao do software.

## Principais funcionalidades

- autenticacao de usuarios com email institucional;
- cadastro, consulta, edicao e visualizacao de pacientes;
- gestao de frequencia de atendimentos;
- upload e consulta de prontuarios;
- criacao e aplicacao de questionarios;
- dashboard com indicadores rapidos;
- administracao de usuarios e permissoes.

## Tecnologias utilizadas

### Backend

- Node.js
- Express
- MySQL
- bcryptjs
- multer

### Frontend

- HTML
- CSS
- JavaScript Vanilla
- live-server para desenvolvimento local

## Escopo atual do sistema

O sistema esta estruturado em frontend e backend separados, com comunicacao HTTP local e persistencia em banco MySQL. A aplicacao contempla os fluxos principais de autenticacao, pacientes, frequencia, prontuarios, questionarios, atividades e administracao de usuarios.
