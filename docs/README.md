# Documentacao do CAIGE

Esta pasta centraliza a documentacao oficial do sistema CAIGE para leitura tecnica, operacao do sistema e entrega academica.

## Objetivo

Organizar o projeto de forma que uma pessoa que nao participou do desenvolvimento consiga:

- entender o problema resolvido pelo sistema;
- instalar o ambiente localmente;
- executar backend, frontend e banco de dados;
- utilizar as funcionalidades principais;
- manter ou evoluir a aplicacao com seguranca;
- consultar referencias tecnicas durante a entrega final.

## Ordem recomendada de leitura

- Para avaliacao academica: visao geral, arquitetura, banco de dados e limitacoes.
- Para desenvolvedores: instalacao, arquitetura, banco de dados, API e implantacao.
- Para usuarios operacionais: manual do usuario e manual do administrador.

## Observacao para a entrega final

Para a versao final do curso, recomenda-se complementar estes documentos com:

- capturas de tela das principais telas do sistema;
- diagramas exportados em PDF ou imagem;
- evidencias de testes;
- versao final das credenciais e instrucoes de acesso;
- historico resumido de alteracoes relevantes.
