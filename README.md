# CAIGE Sistema

Sistema web para gestao de usuarios, pacientes, frequencia, questionarios, cursos e atividades no contexto do CAIGE.

## Stack

- Backend: Node.js, Express e MySQL
- Frontend: HTML, CSS e JavaScript

## Modulos principais

- Autenticacao e perfil
- Gestao de usuarios
- Gestao de pacientes
- Frequencia
- Questionarios e respostas
- Cursos
- Atividades e dados de painel
- Arquivo (usuarios/pacientes arquivados)
