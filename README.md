# CAIGE Sistema

Sistema web para gerenciamento de idosos, atividades, frequencia e prontuarios.

## Documentacao central

Toda a documentacao principal do projeto foi organizada na pasta [docs/README.md](docs/README.md).

## Visao geral

- Backend: Node.js + Express + MySQL
- Frontend: HTML, CSS e JavaScript (Vanilla)
- Execucao local: backend na porta 3000 e frontend na porta 5500

## Como iniciar

Use o guia rapido em [COMO_RODAR.md](COMO_RODAR.md).

## Estrutura principal

```
Backend/
	src/
	database/
	uploads/

Frontend/
	pages/
	assets/
```
